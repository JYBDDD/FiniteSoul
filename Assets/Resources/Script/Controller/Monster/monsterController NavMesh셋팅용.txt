MonsterController 셋팅

크리쳐 애니메이터 파라미터
float MoveZ
float RandA
Bool Die

IdleState()
-> RandA 의 값을 4초마다 (0,-1) 사이로 변경 시켜주고,
    8초에 한번씩 WalkState 상태로 전환시켜 정찰하도록 설정할것

WalkState()
-> Random으로 설정된 정찰위치로 이동한다
    이동 도중 타겟(플레이어)를 발견하지 못한다면 Idle상태로 변경한다

RunningState()
->몬스터에게 시야각 -> viewingAngle
              시야거리 -> viewingDistance 

    설정을 해서 시야에 닿았을 경우 Run 애니메이션 출력후 쫒아가는것으로 설정

AttackState()
->RandA 가 -0.5 ~ -1 -> Attack1
    RandA 가 -0.01 ~ -0.49 -> Attack2

    Mutant 의 공격은 파라미터 불값으로 체크하지 않고
    공격 사정거리안에 플레이어가 접촉하였을경우 실행

HurtState()
->Hurt는 뒤로 밀리는것으로 처리할 예정
        // Idle상태 유지중에서 밀리는것

DieState()
-> 체력이 전부 소진되었을 경우 실행
